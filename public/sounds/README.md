# Arquivos de Áudio para Notificações

Esta pasta deve conter os seguintes arquivos de áudio em formato MP3:

## Arquivos Necessários:

1. **novo-pedido.mp3** - Tocado quando novo pedido chega na cozinha
2. **pedido-pronto.mp3** - Tocado quando pedido fica pronto
3. **alerta.mp3** - Som de alerta geral
4. **sucesso.mp3** - Som de confirmação de ação bem-sucedida

## Formato:
- Formato: MP3
- Duração recomendada: 1-3 segundos
- Qualidade: 128kbps é suficiente

## Observações:
Os arquivos vazios (.mp3) foram criados para evitar erros de carregamento, mas não produzirão som.
Substitua-os por arquivos MP3 reais para ter notificações sonoras funcionais.
